
Hopp over første steg hvis du allerede har sjekket ut repoet:

```
git clone https://github.com/kmkr/react-ws
cd react-ws
git stash
git pull
git stash apply
cd tasks/task-7-state-events
npm install
cd ../task-8-webpack
npm install
cd ../task-9-redux
npm install
```

For å se presentasjonen, åpne `open-presentation.html`
